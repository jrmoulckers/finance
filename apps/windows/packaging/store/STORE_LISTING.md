## Full Description (max 10,000 chars)

**Finance** is a powerful personal finance tracker built natively for Windows. Manage your accounts, track transactions, set budgets, and monitor savings goals — all from a beautiful, fast desktop application.

### Key Features

📊 **Dashboard Overview**
Get a complete picture of your finances at a glance. See net worth, daily and monthly spending totals, recent transactions, and budget health indicators — all in one view.

💳 **Account Management**
Track checking, savings, credit card, investment, and cash accounts. Master-detail layout lets you see account details and transaction history side by side.

📝 **Transaction Tracking**
Full-featured transaction table with search, type filtering (expenses, income, transfers), and sortable columns. Right-click context menus for quick actions.

💰 **Budget Monitoring**
Create budgets for any category and watch your spending in real-time. Animated progress rings show utilization at a glance, with color-coded health indicators (green/amber/red).

🎯 **Savings Goals**
Set financial goals with target amounts and deadlines. Track progress with visual progress bars and monthly contribution estimates.

🔒 **Windows Hello Security**
Protect your financial data with Windows Hello biometric authentication (fingerprint, face recognition, or PIN). Credentials encrypted with Windows DPAPI — never stored in plaintext.

♿ **Full Accessibility**
Built with Narrator screen reader support from the ground up. Every screen, button, and data element has semantic descriptions. Full keyboard navigation (Ctrl+1 through Ctrl+6 for screen switching). High contrast theme support.

⌨️ **Keyboard-First Design**
Navigate with Ctrl+1-6 for screens, Tab for focus movement, right-click for context menus. Every action is reachable without a mouse.

🔄 **Cloud Sync (Coming Soon)**
Sync your financial data across devices with end-to-end encryption. Changes made on any device appear everywhere.

### Why Finance?

- **Native Windows Experience** — Built with Compose Desktop, not a web wrapper. Fluent Design spacing and typography. Segoe UI font.
- **Privacy-First** — Your data is encrypted with DPAPI. No ads, no tracking, no data selling.
- **Fast & Lightweight** — Starts instantly, uses minimal memory.
- **Offline-First** — Works without internet. Sync when you're ready.
